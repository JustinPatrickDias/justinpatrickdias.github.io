Installation

Get 'em runnin'

  1. Download the .jsx file
  2. In After Effects, go to File → Scripts → Install ScriptUI Panel...
  3. Select the .jsx file.
  4. After Effects will ask to restart — go ahead and restart it.
  5. Find it under Window → followed by the script's filename.
